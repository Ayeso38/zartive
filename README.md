## Quick Start

1. Extract the ZIP file
2. Open terminal in that folder  
3. Run: `npm install`
4. Run: `npm run dev`
5. Open http://localhost:5173

## Publishing (Free Options)

### Vercel (Easiest)
1. Go to vercel.com and sign up
2. Run `npm run build` in your project
3. Drag the `dist` folder to Vercel
4. Get your free .vercel.app URL!

### Netlify
1. Go to netlify.com and sign up
2. Run `npm run build`
3. Drag the `dist` folder to Netlify
4. Get your free .netlify.app URL!

### GitHub Pages
1. Create a GitHub repo
2. Push your code
3. Enable Pages in Settings
4. Get username.github.io URL

## Customization

- **Colors**: Edit `index.html` tailwind config
- **Photos**: Replace URLs in `components/Gallery.tsx`
- **Pricing**: Edit `components/Pricing.tsx`
- **About**: Edit the about section in `App.tsx`
